## Commit and Push Guidelines

When working with this repository, do not include any Claude-related references in commits, branches, pull requests, or pushed changes.

Avoid references such as:

- `Claude`
- `Generated with Claude`
- `Co-Authored-By: Claude`
- AI-generated attribution footers
- Claude-specific comments in source files
- Claude-related branch names or commit messages

Commit messages should describe the actual code or documentation change only.

Before committing or pushing, check that no Claude attribution, metadata, or generated signature has been added to the repository.
